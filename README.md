DMart Sales Data Analysis – Advanced Excel

Project Overview

This project focuses on exploring and understanding a DMart sales dataset using Microsoft Excel. The dataset contains over 15,000 sales records and includes important business fields such as sales, profit, discount, quantity, customer type, payment mode, region, category, and city.

Dataset Fields

- Order ID
- Region
- Category
- Sales
- Discount
- Quantity
- Order Date
- Delivery Date
- Customer Type
- Payment Mode
- Profit
- City

Objectives

- Explore and understand the structure of the dataset.
- Analyze key sales-related fields.
- Identify important business metrics.
- Prepare the dataset for further analysis and dashboard creation.

- Data Exploration

- Basic Excel Functions
